# Инструкция по развертыванию веб-приложения «Аптека Здоровье+»

Данное руководство описывает пошаговый процесс развертывания проекта на операционной системе Ubuntu. Проект состоит из двух частей: Backend (Java/Spring Boot) и Frontend (Node.js/Express).

## 1. Системные требования

Для успешного запуска проекта на сервере (или локальном компьютере) с ОС Ubuntu должны быть установлены следующие компоненты:
* **Java Development Kit (JDK) 21** (для компиляции и запуска бэкенда)
* **Node.js** (версия 18 или выше) и пакетный менеджер **npm** (для запуска фронтенда)

## 2. Установка зависимостей в Ubuntu

Откройте терминал и выполните следующие команды для установки необходимых пакетов.

### Установка Java (JDK 21)
```bash
sudo apt update
sudo apt install openjdk-21-jdk -y
```
Проверьте установку:
```bash
java -version
```

### Установка Node.js и npm
Рекомендуется устанавливать Node.js через официальный репозиторий NodeSource:
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```
Проверьте установку:
```bash
node -v
npm -v
```

## 3. Подготовка и запуск Backend (Серверная часть API)

Бэкенд приложения написан на Java с использованием фреймворка Spring Boot. В качестве базы данных используется встраиваемая In-Memory БД H2, поэтому дополнительная установка сервера баз данных (например, PostgreSQL или MySQL) не требуется.

1. Перейдите в директорию бэкенда:
   ```bash
   cd /путь/к/проекту/backend
   ```
2. Проект использует Maven Wrapper (`mvnw`), поэтому устанавливать Maven отдельно не нужно. Соберите и запустите проект с помощью команды:
   ```bash
   ./mvnw spring-boot:run
   ```
   *При первом запуске Maven автоматически скачает все необходимые Java-библиотеки.*

3. Убедитесь, что сервер успешно запустился. В консоли должно появиться сообщение `Started BackendApplication in ... seconds`.
   Сервер запустится на порту **8080** (`http://localhost:8080`).

## 4. Подготовка и запуск Frontend (Пользовательский интерфейс)

Фронтенд написан на Node.js и использует Express для рендеринга HTML-страниц (шаблонизатор EJS).

1. Откройте **новое окно терминала** (так как в первом работает бэкенд).
2. Перейдите в директорию фронтенда:
   ```bash
   cd /путь/к/проекту/frontend
   ```
3. Установите необходимые NPM-зависимости (Express, Axios, EJS и др.):
   ```bash
   npm install
   ```
4. Запустите веб-сервер:
   ```bash
   node app.js
   ```
5. Сервер запустится на порту **3000**. В консоли появится сообщение: `Frontend server running on port 3000`.

## 5. Использование приложения

* Откройте браузер и перейдите по адресу: **http://localhost:3000**
* **Аккаунт Администратора** (создается автоматически при первом запуске):
  * Логин: `admin`
  * Пароль: `admin123`
* Обычные пользователи могут зарегистрироваться через раздел "Регистрация" в меню сайта.

### Решение возможных проблем
* **Проблема: "Port 8080 / 3000 is already in use"**
  Это означает, что порты заняты другим приложением. Вы можете найти и завершить процесс с помощью команды:
  ```bash
  sudo fuser -k 8080/tcp
  sudo fuser -k 3000/tcp
  ```
* **Проблема с правами доступа к `mvnw`:**
  Если при запуске `./mvnw` появляется ошибка *Permission denied*, выдайте файлу права на исполнение:
  ```bash
  chmod +x mvnw
  ```
